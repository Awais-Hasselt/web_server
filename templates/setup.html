  • When powering the barrel you will hear one beep.
  • If this is the first time using the barrel, you will hear two beeps in a few seconds. This indicates that its setup mode is started.
  • If you configured your barrel before, and you want to start the setup mode manually to reconfigure it, then hold the button on the barrel while powering it. Hold this until it beeps twice, indicating setup mode.
  • When setup mode is started, it will create a WiFi network called awais-connect. Connect to this network. Note that you cannot access the internet while connected.
  • Visit http://awais.local in your web browser (this might take a while to load).
  • Enter your network credentials and a name for the barrel to configure the barrel to use your WiFi (or hotspot). Note: you cannot use enterprise networks with a user and password. Only use networks with just an SSID and password.
  • After saving your credentials, the barrel will restart (and the awais-connect network will disappear) and try to connect to your WiFi.
  • If you hear three beeps, it has connected. If it cannot connect, you will hear five beeps.
